## üìå Overview  
This project contains an end-to-end **AI-powered automation agent** that:

- Downloads a file from Google Drive  
- Extracts text from a PDF  
- Processes the content using an LLM-based AI Agent  
- Formats the result with JavaScript  
- Logs the structured output to Google Sheets  

The workflow provides a seamless **PDF ‚Üí AI Analysis ‚Üí Google Sheet** pipeline.

### **1Ô∏è‚É£ Execute Workflow Trigger**  
The automation begins when the user starts the workflow.

---

### **2Ô∏è‚É£ Download File from Google Drive**  
- Retrieves the specified PDF file through a direct link  
- Ensures a valid and up-to-date copy is used for processing  

---

### **3Ô∏è‚É£ Extract Text from PDF**  
- Uses a PDF extractor to pull clean, readable text

### **4Ô∏è‚É£ AI Agent Processing (LLM)**

The AI Agent:

- Generates summaries or key insights  
- Ensures clean, context-aware output

### **5Ô∏è‚É£ JavaScript Formatting**
